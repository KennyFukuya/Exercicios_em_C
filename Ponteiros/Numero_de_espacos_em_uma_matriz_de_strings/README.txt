Leia um inteiro N, o qual representa o número de linhas a serem lidas. 
O comprimento máximo de cada linha é 100 carácteres. ( talvez nao tenhamos casos de teste ateh 100) Sugestão: 
As linhas serão lidas como entrada, armazene-as em uma matriz N x 100

Atividade:

Crie um vetor de N ponteiros.
Cada ponteiro aponta para UMA linha da matriz acima.
Crie uma função que conte o número de espaços em branco.
Usando os N vetores, encontre quantos espaços em branco existem nesta matriz
Retorne o número de espaços em branco.
