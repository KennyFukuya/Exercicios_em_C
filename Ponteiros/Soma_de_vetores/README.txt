Faça uma função que receba por parâmetro dois vetores de inteiros (v1 e v2) e um inteiro (n) que representa o tamanho de cada vetor. 
A função deve retornar um vetor resultante (vr) da soma de cada elemento. 
Por exemplo:

int *soma_vetores(int *v1, int *v2, int n) ;

vr[0] = v1[0] + v2[0]

vr[1] = v1[1] + v2[1]

...

vr[n] = v1[n] + v2[n]

A primeira linha de entrada do programa é um valor inteiro n, que representa o tamanho de cada vetor. 
As 2 próximas linhas da entrada contém, cada uma, n inteiros, que preenchem o vetores v1 e v2. 
A saída são n inteiros, separados por espaço, que representam o vetor resultante da soma dos vetores.

Obs.: v1 e v2 sempre terão o mesmo tamanho (n).
